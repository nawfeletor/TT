# 🦇 Nawfy

**A personal music player that streams YouTube audio directly.**  
No browser. No ads. No subscription. Open source, free forever.

> Built with Electron + yt-dlp · Goth dark theme · Windows & Linux

## 🚀 Quick Start

```bash
git clone https://github.com/YOUR_USERNAME/nawfy.git
cd nawfy
npm install
npm start           # dev
npm run build:win   # Windows installer + portable
npm run build:linux # AppImage + .deb
```

**Node.js 18+** required. On first launch click "Download yt-dlp" — one-time setup (~18 MB).
